# Library Management System

A simple console-based Library Management System developed in Python, following specific functional flowcharts.

## Features
1. **Add Book** – Input and store new book details.
2. **Register Member** – Register new library members.
3. **Borrow Book** – Process book borrowing with validation.
4. **View Books** – Display all books with availability status.
5. **View Members** – Display all registered members.
6. **View Loans** – Display all borrowing records.
7. **Exit** – Terminate the program.

## How to Run
1. Ensure Python is installed.
2. Run the script:
```bash
python library_management.py
